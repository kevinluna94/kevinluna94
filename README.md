# 4) Plantillas de Issues y Pull Requests

Crea la carpeta `.github/ISSUE_TEMPLATE/` y agregá estos archivos:

**`.github/ISSUE_TEMPLATE/bug_report.yml`**

```yml
name: Bug report
description: Reportá un error
labels: [bug]
body:
  - type: textarea
    id: describe
    attributes:
      label: ¿Qué pasa?
      description: Describí el bug y pasos para reproducir
    validations:
      required: true
  - type: input
    id: expected
    attributes:
      label: ¿Qué esperabas que ocurra?
  - type: input
    id: env
    attributes:
      label: Entorno
      placeholder: OS, versión, navegador, etc.
```

**`.github/ISSUE_TEMPLATE/feature_request.yml`**

```yml
name: Feature request
description: Proponé una mejora
labels: [enhancement]
body:
  - type: textarea
    id: value
    attributes:
      label: Valor de negocio
      description: ¿Qué mejora y por qué?
      placeholder: KPI, tiempo ahorrado, etc.
    validations:
      required: true
  - type: textarea
    id: solution
    attributes:
      label: Posible solución
```

**`PULL_REQUEST_TEMPLATE.md`** (en `.github/`)

# 5) GitHub Actions (CI)

## a) CI para proyectos **Node/React/Vite** — `.github/workflows/ci-node.yml`

```yml
name: CI (Node)
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  build-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm
      - run: npm ci
      - run: npm run lint --if-present
      - run: npm test --if-present
      - run: npm run build --if-present
```

## b) CI para proyectos **Python** — `.github/workflows/ci-python.yml`

```yml
name: CI (Python)
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - run: |
          python -m venv .venv
          source .venv/bin/activate
          pip install -r requirements.txt
          pip install black flake8 pytest
          black --check .
          flake8 .
          pytest -q
```
